# Neural Network in C++

Because why not?

This project is a simple neural network built from scratch in C++. The goal is not to compete with PyTorch or TensorFlow, but to understand what is actually happening underneath: matrices, weights, biases, activation functions, and forward propagation.

## Goals

- Build a minimal matrix class using `std::vector`
- Implement basic matrix operations
- Represent neural network layers with weights and biases
- Run a simple forward pass (probably a linear one)
- Eventually support training with backpropagation
